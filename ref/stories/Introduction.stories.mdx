# STM32 AI 嵌入式编程平台组件库

欢迎使用STM32 AI嵌入式编程平台组件库！这个组件库包含了平台中使用的所有UI组件和功能组件，为开发人员提供了一致的设计语言和可复用的组件。

## 组件分类

组件库按照功能和用途分为以下几类：

### UI组件

基础UI组件，用于构建用户界面的基本元素。

- **Button** - 按钮组件，支持多种变体、尺寸和状态
- **Card** - 卡片组件，用于展示内容的容器
- **Alert** - 警告提示组件，用于显示提示信息
- **Badge** - 徽章组件，用于显示状态或计数
- **Input** - 输入框组件，用于收集用户输入
- **Select** - 下拉选择组件，用于从多个选项中选择
- **Checkbox** - 复选框组件，用于多选场景
- **Radio** - 单选框组件，用于单选场景
- **Switch** - 开关组件，用于切换状态
- **Modal** - 模态对话框组件，用于显示重要信息或收集用户输入
- **Tooltip** - 工具提示组件，用于显示额外信息
- **Tabs** - 标签页组件，用于内容分组显示
- **Dropdown** - 下拉菜单组件，用于显示菜单选项
- **Pagination** - 分页组件，用于分页显示数据
- **Progress** - 进度条组件，用于显示进度
- **Spinner** - 加载中组件，用于显示加载状态

### 布局组件

用于构建页面布局的组件。

- **MainLayout** - 主布局组件，包含顶部导航栏、侧边栏和内容区域
- **AuthLayout** - 认证布局组件，用于登录、注册等认证页面
- **Sidebar** - 侧边栏组件，用于导航
- **Navbar** - 导航栏组件，用于顶部导航
- **Footer** - 页脚组件，用于显示页脚信息
- **Container** - 容器组件，用于限制内容宽度
- **Grid** - 网格组件，用于构建响应式布局
- **Flex** - 弹性布局组件，用于构建弹性布局
- **Box** - 盒子组件，用于构建基本布局

### 卡片组件

用于展示不同类型内容的卡片组件。

- **CourseCard** - 课程卡片组件，用于展示课程信息
- **ExperimentCard** - 实验卡片组件，用于展示实验信息
- **ChallengeCard** - 挑战卡片组件，用于展示挑战信息
- **ResourceCard** - 资源卡片组件，用于展示资源信息
- **HomeCourseCard** - 首页课程卡片组件，用于在首页展示课程信息

### 图表组件

用于展示数据可视化的组件。

- **MermaidFlowchart** - Mermaid流程图组件，用于展示流程图
- **MermaidToReactFlow** - Mermaid转ReactFlow组件，用于将Mermaid语法转换为ReactFlow组件
- **ReactFlowNodes** - ReactFlow节点组件，用于自定义ReactFlow节点
- **FlowchartContainer** - 流程图容器组件，用于包装流程图组件

### 功能组件

提供特定功能的组件。

- **WebIDEContainer** - WebIDE容器组件，用于包装WebIDE功能
- **SerialDebuggerContainer** - 串口调试器容器组件，用于包装串口调试器功能
- **ErrorDebuggerContainer** - 错误调试器容器组件，用于包装错误调试器功能
- **CodeGeneratorContainer** - 代码生成器容器组件，用于包装代码生成器功能

## 使用指南

### 安装

组件库已经集成在项目中，无需额外安装。

### 导入组件

使用以下方式导入组件：

```jsx
// 从根索引文件导入
import { Button, Card, Alert } from '../components';

// 从分类索引文件导入
import { Button, Card, Alert } from '../components/ui';
import { CourseCard, ExperimentCard } from '../components/cards';
import { MermaidFlowchart } from '../components/charts';
```

### 示例

```jsx
import React from 'react';
import { Button, Card } from '../components/ui';

function Example() {
  return (
    <Card>
      <Card.Header>
        <Card.Title>示例卡片</Card.Title>
      </Card.Header>
      <Card.Body>
        这是一个示例卡片，展示了如何使用Card组件。
      </Card.Body>
      <Card.Footer>
        <Button variant="primary">确认</Button>
        <Button variant="outline" className="ml-2">取消</Button>
      </Card.Footer>
    </Card>
  );
}
```
